Breve linea del tiempo: Programacion

  • 1801: El telar de Jacquard, considerado el primer dispositivo programable.
  • 1833: Ada Lovelace escribe el primer algoritmo para ser procesado por una máquina, considerada la primera programadora.
  • 1936: Alan Turing propone la Máquina de Turing, un modelo teórico para la computación.
  • 1945: Se construye la ENIAC, la primera computadora electrónica de propósito general.
  • 1954: John Backus desarrolla el primer lenguaje de programación de alto nivel, FORTRAN.
  • 1972: Dennis Ritchie crea el lenguaje de programación C.
  • 1978: Se lanza el primer lenguaje de programación orientado a objetos, Smalltalk.
  • 1991: Linus Torvalds crea el kernel de Linux, un sistema operativo de código abierto.
  • 1995: Java es lanzado por Sun Microsystems, popularizando la programación orientada a objetos.
  • 2004: Mark Zuckerberg crea Facebook, impulsando el desarrollo de aplicaciones web y redes sociales.
  • 2007: Apple lanza el iPhone, revolucionando la industria móvil y creando nuevas oportunidades de desarrollo.
  • 2010: Se lanza Python 3.0, una versión importante del popular lenguaje de programación.
  • 2021: Elon Musk funda Neuralink, una empresa que busca fusionar la inteligencia artificial con el cerebro humano.
